# Jupyter Notebooks

## 说明

Jupyter notebooks用于交互式分析和可视化。

## 待提供的notebooks（第五批）

1. `00_environment_test.ipynb` - 环境测试
2. `01_data_exploration.ipynb` - 数据探索和统计
3. `02_feature_analysis.ipynb` - 特征分析
4. `03_model_comparison.ipynb` - 模型性能对比
5. `04_visualization.ipynb` - 结果可视化
6. `05_final_report.ipynb` - 最终报告生成

## 使用方法

```bash
# 启动Jupyter
jupyter notebook

# 在浏览器中打开notebooks
```

## 依赖

确保已安装：
- jupyter
- ipykernel
- matplotlib
- seaborn
- plotly
